# Installation Instructions

## Load the extension

1. Open Chrome and go to `chrome://extensions/`.
2. Enable `Developer mode`.
3. Click `Load unpacked`.
4. Select the extension folder.
5. If ChatGPT was already open, refresh that tab once.

## Confirm it is working

1. Open `https://chatgpt.com`.
2. Trigger any tool that requires approval (Desktop Commander, an MCP server, browser tool, etc.).
3. Leave `Auto-approve` enabled and confirm the prompt is clicked automatically.
4. Open the popup if you want to verify the watcher status, recent approvals, or settings.

## Troubleshooting

- If the popup says the scanner is not ready, refresh the ChatGPT tab once.
- If no approvals are being clicked, confirm `Auto-approve` is enabled in the popup settings.
- If the badge looks stale, toggle `Badge count` off and back on or reload the ChatGPT tab.
